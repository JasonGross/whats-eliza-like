Eliza asks: What's that like?

ELIZA is a natural language conversation program described by Joseph Weizenbaum in January 1966. It features the dialog between a human user and a computer program representing a mock Rogerian psychotherapist.

This instantiation is based on elizabot.js by © Norbert Landsteiner 2005 at http://www.masswerk.at/elizabot.
The implementation is, at present, largely unchanged from that one, while the data driving what they system says was written by Jason Gross.

The source for this project is available on GitHub.
